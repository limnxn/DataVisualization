# Data Visualization

## Introduction

You can clone this repository to try a data exploration and visualization platform.

## First

### If you have a dump file

```bash
$ cp yourdumpfile mysql/initdb.d
```
## Docker

### Start app containers

```bash
$ docker compose up -d
```

- `3000` Grafana
- `3030` Metabase
- `8080` Superset
